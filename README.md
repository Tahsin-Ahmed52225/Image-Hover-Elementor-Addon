# Image-Hover-Plugin V1.0.1
## This is a WordPress Elementor addon which has two image hover animation 
1. Scroll Up image when hover 
2. Scroll Down Image when hover 

Functionalities : 
> Adding Title and Customize it <br>
> Remove Title <br>
> Adding Icon <br>

Requirments :
> Wordpress <br>
> Elementor <br>
